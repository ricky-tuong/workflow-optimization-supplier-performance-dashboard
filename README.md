# Workflow Optimization & Supplier Performance Excel Project

This project demonstrates **workflow optimization and supplier performance analysis** using Excel Web. It showcases key metrics, pivot tables, supporting tables, and a professional dashboard for decision-making.

The project simulates a **real-world business environment**, helping to track supplier performance, approval status, missing documents, and contract metrics.

---

## Project Objective
Create an **interactive, easy-to-read Excel dashboard** that summarizes supplier activity and identifies workflow bottlenecks, enabling informed business decisions.

---

## Key Features

### 1. Raw Data
- Contains all original requests and supplier information.  
- Serves as the source for all pivot tables and charts.

### 2. Pivot Tables
- **Supplier Performance:** Count of Requests, Average Contract Value, Average Lead Time.  
- **Approval Status:** Tracks Approved, Pending, and Rejected requests.  
- **Missing Documents:** Identifies suppliers with incomplete documentation.  
- **Risk Levels:** Summarizes supplier risk based on defined criteria.

### 3. Supporting Tables
- Clean, structured tables derived from pivot tables.  
- Feed charts for the dashboard.  
- Includes static and dynamic tables for analysis.

### 4. Dashboard
- Interactive visual summary of key metrics.  
- Charts include:
  - Supplier Performance (requests, contract value, lead time)  
  - Approval Status (bar chart)  
  - Missing Documents (color-coded bar chart)  
  - Risk Levels (pie chart)  
  - Top 5 Suppliers (highlighted chart)  
- Charts use **consistent color coding and alphabetical supplier order** for readability.

---

## Tools & Techniques
- **Excel Web:** Pivot tables, charts, conditional formatting, and static tables.  
- **Workflow Optimization Concepts:** Process visibility, supplier tracking, KPI analysis.  
- **Data Visualization:** Clean, readable charts for portfolio and business presentation.

---

## How to Use
1. Open `workflow_optimization_supplier_performance.xlsx` in Excel Web or desktop Excel.  
2. Refresh pivot tables to update data if raw data changes.  
3. Use charts and supporting tables to analyze supplier performance and workflow efficiency.
